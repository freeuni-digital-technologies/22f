рЃЌрЃљрЃАрЃЦрЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ

рЃЊрЃљрЃљрЃЎрЃЋрЃўрЃарЃЊрЃў, рЃарЃЮрЃЏ рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃЮрЃарЃў рЃ«рЃљрЃќрЃў рЃўрЃњрЃўрЃЋрЃћ рЃарЃЕрЃћрЃЉрЃљ 

function addTask() {
    // рЃерЃћрЃДрЃЋрЃљрЃюрЃўрЃџрЃў рЃбрЃћрЃЦрЃАрЃбрЃўрЃА рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљ
    let taskText = document.getElementById('task-input').value

    // рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ
    let taskElement = document.createElement('div')
    taskElement = taskText

    // рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ tasks рЃЊрЃўрЃЋрЃерЃў 
    let tasksDiv = document.getElementById('tasks')
    tasksDiv.appendChild(taskElement)
}

рЃарЃћрЃцрЃљрЃЦрЃбрЃЮрЃарЃўрЃюрЃњрЃў

рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃюрЃљрЃЉрЃўрЃ»рЃўрЃАрЃЌрЃЋрЃўрЃА рЃњрЃЋрЃўрЃюрЃЊрЃљ рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџ рЃЌрЃљрЃАрЃЦрЃА рЃЦрЃЮрЃюрЃЊрЃћрЃА рЃдрЃўрЃџрЃљрЃЎрЃў, рЃарЃЮрЃЏрЃџрЃўрЃЌрЃљрЃф рЃЏрЃљрЃА рЃерЃћрЃАрЃарЃБрЃџрЃћрЃЉрЃБрЃџрЃљрЃЊ рЃЏрЃЮрЃЋрЃюрЃўрЃерЃюрЃљрЃЋрЃЌ рЃ«рЃЮрЃџрЃЏрЃћ. addTask() рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ рЃБрЃЎрЃЋрЃћ рЃАрЃљрЃЎрЃЏрЃљрЃарЃўрЃАрЃў рЃќрЃЮрЃЏрЃўрЃАрЃљрЃљ рЃЊрЃљ рЃЎрЃўрЃЊрЃћрЃЋ рЃЏрЃћрЃбрЃў рЃЎрЃЮрЃЊрЃўрЃА рЃЕрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌ рЃљрЃўрЃарЃћрЃЋрЃљ. рЃљрЃЏрЃўрЃбрЃЮрЃЏ, рЃгрЃўрЃюрЃљрЃАрЃгрЃљрЃарЃЋрЃћ рЃЏрЃЮрЃЋрЃљрЃЏрЃќрЃљрЃЊрЃЮрЃЌ рЃљрЃ«рЃљрЃџрЃў рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ.

рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃўрЃљ, рЃарЃЮрЃЏ рЃЌрЃБ рЃљрЃ«рЃљрЃџрЃў рЃцрЃБрЃюрЃЦрЃфрЃўрЃўрЃАрЃљрЃА рЃарЃћрЃцрЃљрЃЦрЃбрЃЮрЃарЃўрЃюрЃњрЃўрЃА рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃџрЃЮрЃЉрЃљрЃА рЃЏрЃўрЃ«рЃЋрЃЊрЃў, рЃ»рЃћрЃа рЃљрЃарЃАрЃћрЃЉрЃБрЃџ рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃќрЃћ рЃњрЃљрЃљрЃЎрЃћрЃЌрЃЮ рЃарЃћрЃцрЃљрЃЦрЃбрЃЮрЃарЃўрЃюрЃњрЃў.

рЃљрЃ«рЃљрЃџрЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ

function createTaskElement(taskText) {
    let myDiv = document.createElement('div')
    myDiv.innerText = taskText
    myDiv.className = 'task-div'
    return myDiv
}

рЃњрЃљрЃ«рЃАрЃћрЃюрЃў рЃЎрЃЮрЃюрЃАрЃЮрЃџрЃў рЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃўрЃФрЃљрЃ«рЃћ createTaskElement() рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ. рЃљрЃарЃњрЃБрЃЏрЃћрЃюрЃбрЃљрЃЊ рЃарЃљрЃЏрЃћ рЃАрЃбрЃарЃўрЃюрЃњрЃў рЃњрЃљрЃЊрЃљрЃљрЃгрЃЮрЃЊрЃћ, рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ createTaskElement("mytask"). рЃерЃћрЃЏрЃЊрЃћрЃњ рЃЎрЃў рЃЏрЃЌрЃљрЃЋрЃљрЃа рЃцрЃБрЃюрЃЦрЃфрЃўрЃљрЃерЃўрЃф рЃњрЃљрЃЏрЃЮрЃўрЃФрЃљрЃ«рЃћ

рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ

рЃерЃћрЃЋрЃЦрЃЏрЃюрЃљрЃЌ рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃАрЃљрЃф рЃњрЃљрЃЊрЃљрЃЋрЃљрЃгрЃЋрЃЊрЃўрЃЌ рЃарЃљрЃЏрЃћ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃА, рЃЊрЃљ рЃЏрЃљрЃА рЃЊрЃљрЃљрЃЏрЃљрЃбрЃћрЃЉрЃА рЃЌрЃљрЃАрЃЦрЃћрЃЉрЃўрЃА рЃЊрЃўрЃЋрЃерЃў 

function addTaskElement(element) {
    document.getElementById('tasks').appendChild(element)
}

рЃњрЃљрЃ«рЃАрЃћрЃюрЃў рЃЎрЃЮрЃюрЃАрЃЮрЃџрЃў рЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃўрЃФрЃљрЃ«рЃћ createTaskElement() рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ, рЃЮрЃдрЃЮрЃюрЃЊ рЃљрЃЏрЃ»рЃћрЃарЃљрЃЊ рЃЏрЃўрЃАрЃў рЃЊрЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃБрЃџрЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ рЃерЃћрЃўрЃюрЃљрЃ«рЃћ, рЃЊрЃљ рЃерЃћрЃЏрЃЊрЃћрЃњ рЃЌрЃљрЃАрЃЦрЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃА рЃцрЃБрЃюрЃЦрЃфрЃўрЃљрЃА рЃњрЃљрЃЊрЃљрЃљрЃгрЃЮрЃЊрЃћ.

> let taskElement = createTaskElement("my task")
> taskElement // рЃЊрЃљрЃарЃгрЃЏрЃБрЃюрЃЊрЃў, рЃарЃЮрЃЏ рЃфрЃЋрЃџрЃљрЃЊрЃерЃў dom рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃљ рЃерЃћрЃюрЃљрЃ«рЃБрЃџрЃў
> addTaskElement(taskElement) // рЃљрЃ«рЃџрЃљ рЃЎрЃў

рЃћрЃА рЃ«рЃљрЃќрЃћрЃЉрЃў рЃерЃћрЃЏрЃЊрЃћрЃњрЃўрЃА рЃћрЃЎрЃЋрЃўрЃЋрЃљрЃџрЃћрЃюрЃбрЃБрЃарЃўрЃљ: 

addTaskElement(createTaskElement("my task"))

рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃцрЃБрЃюрЃЦрЃфрЃўрЃўрЃА рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃљ

рЃљрЃ«рЃџрЃљ рЃЊрЃарЃЮрЃљ рЃЮрЃарЃў рЃљрЃ«рЃљрЃџрЃў рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ рЃАрЃгрЃЮрЃарЃљрЃЊ рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃЮрЃЌ 

function addTask() {
    let taskText = document.getElementById('task-input').value
    let taskElement = createTaskElement(taskText)
    addTaskElement(taskElement)
}
